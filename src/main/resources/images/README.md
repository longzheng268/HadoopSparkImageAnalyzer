# 图像资源目录

此目录用于存放分析用的图像文件。

## 如何获取图像资源

### 方法1：使用内置下载功能（推荐）
运行程序后，在GUI界面点击"下载样本图像"按钮，程序会自动从 Lorem Picsum 下载灰度图像。

### 方法2：手动添加图像
将您自己的图像文件（.jpg, .jpeg, .png, .bmp）直接复制到此目录。

### 方法3：使用专业数据集
下载 BOSSBase 数据集并解压到此目录：
- 下载地址：http://agents.fel.cvut.cz/boss/index.php?mode=VIEW&tmpl=materials

## 图像规格建议
- 尺寸：512×512 像素
- 格式：JPEG、PNG、BMP
- 类型：灰度图像（黑白）

## 注意事项
- 下载的图像文件会被 .gitignore 忽略，不会提交到 Git 仓库
- 建议用于测试时下载 10-50 张图像即可
